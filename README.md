# HelloApp

## Overview
HelloApp is a simple application designed to demonstrate basic programming concepts and GitHub workflow.

## Purpose
The purpose of this project is to implement different use cases for displaying messages such as Hello World and user-specific greetings.

## Project Structure
- src/ → Source code
- bin/ → Compiled binaries
- docs/ → Documentation files

## How to Run

1. Navigate to the src folder
2. Compile the program

javac HelloApp.java

3. Run the program

java HelloApp
